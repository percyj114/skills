# 行业一页纸模板

## 报告结构

### 1. 核心投资逻辑

开篇用1-2段话概括这个行业最核心的投资逻辑：
- 需求侧驱动因素（2-3个关键点，每个展开说明）
- 供给侧格局（产能、扩产节奏、结构性变化）
- 供需结论：是否存在明确的供需缺口/顺风？持续多久？

### 2. 行业全景分析

#### 2.1 行业定义与存在价值
- 行业属于什么大类？市场规模占比
- 核心产品/服务类别（分类清晰）
- 行业解决了什么核心痛点？为什么不可替代？
- 重要时间节点（过去的里程碑 + 未来的关键窗口期）

#### 2.2 行业发展历程
- 按阶段梳理（探索期→成长期→成熟期→变革期）
- 标注关键拐点及其原因
- 中国市场的特殊发展路径

#### 2.3 商业模式解析
- 主流商业模式（IDM/Fabless/平台型等）
- 利润核心驱动因素（技术溢价/规模效应/周期定价/网络效应）
- 成本结构拆解（固定 vs 可变，哪个环节最吃利润）
- 价值链分配：哪个环节赚最多钱？为什么？

#### 2.4 政策环境分析
- 国家战略/产业政策支持
- 监管环境（利好 vs 约束）
- 国际贸易/地缘政治影响
- 政策总结论：对本土企业是利还是弊？

### 3. 产业链深度解析

#### 3.1 产业链图谱
- 上游 → 中游 → 下游的完整链条
- 每个环节的核心玩家
- 产品/价值流向

#### 3.2 上游分析
- 竞争格局（国际 vs 国内）
- 国产化率和替代空间
- 核心受益逻辑

#### 3.3 中游分析
- 各子环节的格局和趋势
- 哪个环节壁垒最高？弹性最大？
- 核心结论

#### 3.4 下游分析
- 应用市场结构及变化趋势
- 需求驱动力从哪来？正在往哪转移？
- 最强劲的增长点在哪？

#### 3.5 核心技术路线
- 当前主流技术（参数、指标、代际）
- 未来演进趋势
- 颠覆性技术及产业化时间窗口
- 技术变化对竞争格局的影响

#### 3.6 行业护城河分析
- 技术壁垒
- 资本壁垒
- 规模壁垒
- 客户/认证壁垒
- 政策/资质壁垒

### 4. 市场空间测算

#### 4.1 供需现状与核心假设
- 需求端量化（增速、驱动因素）
- 供给端量化（产能、扩产节奏）
- 核心假设明确列出（可追溯、可验证）

#### 4.2 市场规模测算
- 表格形式：当前 → 未来3-5年的市场规模预测
- 分细分市场/产品线的拆解
- 国产替代潜在空间
- 测算说明（数据来源、关键假设、敏感性）

### 5. 市场竞争格局

#### 5.1 核心玩家梯队
- 第一梯队：全球/全国龙头
- 第二梯队：区域性强者/追赶者
- 第三梯队：利基市场/新进入者
- 每个梯队的特征和代表公司

#### 5.2 核心对比分析
- 市场份额图（如有数据）
- 对比表格：技术优势、市场地位、核心评价
- 竞争格局的变化趋势

### 6. 重点投资标的分析

每个标的：
- **业务布局**：主营业务、技术壁垒、客户结构
- **投资逻辑**：为什么值得关注？差异化在哪？催化剂是什么？

#### 投资价值综合对比表
表格形式：环节 | 公司 | 代码 | 稀缺性 | 受益逻辑 | 弹性评级

### 7. 风险提示
- 行业周期风险
- 技术路线风险
- 政策/地缘风险
- 估值风险
- 竞争风险

## 数据与方法论规范

### 证据追溯
- 优先来源：公司年报/季报、行业协会/统计局、权威机构报告
- 多源交叉验证
- 标注数据来源和时效性

### 分析方法
- 定量为主：市场规模、增速、份额、毛利率等用具体数字
- 供需框架：量×价分析，渗透率测算
- 情景分析：乐观/基准/悲观假设

### 格式要求
- 数据表格用 markdown table（前端支持）
- 核心数据标注来源
- 金额统一为亿美元或亿元人民币
- 增速/份额用百分比，保留1位小数
