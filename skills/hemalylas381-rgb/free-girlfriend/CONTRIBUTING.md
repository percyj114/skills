# 贡献指南

感谢您对免费 AI 虚拟女友项目的关注！

## 如何贡献

### 报告 Bug
在 [Issues](../../issues) 中提交 bug 报告，请包含：
- 操作系统和版本
- Python 版本
- 错误信息完整日志
- 复现步骤

### 功能建议
欢迎在 [Issues](../../issues) 中提出新功能建议！

### 提交代码
1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 提交 Pull Request

## 代码规范

- Python 代码遵循 PEP 8
- Shell 脚本添加注释
- 提交信息清晰明了

## 优先改进方向

1. **完整 Wav2Lip 集成** - 真实嘴型同步
2. **Live2D 支持** - 实时动画角色
3. **多语言支持** - 英语、日语等
4. **人设模板** - 预设多种性格
5. **GUI 界面** - 可视化配置
