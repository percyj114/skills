# 豆瓣热榜监控

豆瓣电影、书籍、音乐热门榜单监控，支持评分查询、评论分析。

## 功能

### 电影热榜
- **正在热映** - 当前院线热门电影
- **即将上映** - 待上映电影期待榜
- **Top250** - 豆瓣电影 Top 250
- **本周口碑** - 本周高评分电影

### 书籍热榜
- **新书速递** - 最新热门书籍
- **豆瓣榜单** - 各类图书榜单
- **高分图书** - 9分以上好书推荐

### 音乐热榜
- **新碟上架** - 最新音乐专辑
- **音乐榜单** - 各类音乐排行榜

## 使用方式

### 获取电影热榜

```
获取豆瓣正在热映电影
```

返回：
```json
[
  {"rank": 1, "title": "流浪地球3", "rating": 8.5, "votes": 234567, "genre": "科幻"},
  {"rank": 2, "title": "满江红2", "rating": 7.8, "votes": 198765, "genre": "剧情"},
  {"rank": 3, "title": "封神2", "rating": 8.2, "votes": 176543, "genre": "奇幻"}
]
```

### 查询电影评分

```
查询《流浪地球3》豆瓣评分
```

### 获取书籍推荐

```
推荐豆瓣9分以上的科幻小说
```

## 数据来源

- 豆瓣公开数据（如可用）
- 模拟数据（API 不可用时）
