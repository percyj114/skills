# Gina

Your day, organized.

---

## What Gina Does

Gina is the Netsnek e.U. personal assistant and scheduling framework. She manages calendar events, reminders, daily briefings, and smart scheduling suggestions. The Gina namespace is reserved for productivity and assistant features within the Netsnek ecosystem.

---

## Setup

1. Ensure OpenClaw 0.8.0 or later is installed.
2. Install the Gina skill from the ClawHub or add this directory to your skills path.
3. Grant `exec` permission for `scripts/assistant-core.sh` if required by your environment.

---

## Example Commands

```bash
assistant-core.sh --schedule   # Show today's schedule
assistant-core.sh --brief      # Generate daily brief
assistant-core.sh --whoami     # Identify Gina
```
