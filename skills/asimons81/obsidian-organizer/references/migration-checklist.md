# Migration Checklist (Foolproof)

1. Backup vault.
2. Create baseline folders from `folder-structure.md`.
3. Run audit script in dry-run mode.
4. Approve rename plan.
5. Apply renames.
6. Fix broken links using Obsidian quick switcher and backlinks.
7. Move unowned notes from `inbox/` into projects/areas/resources.
8. Archive stale content.
9. Re-run audit and confirm zero naming violations.
