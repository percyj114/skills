# File Naming Rules

Use deterministic naming:

1. Kebab-case only: `lowercase-words.md`
2. No spaces, emojis, or punctuation besides `-`
3. Daily notes: `YYYY-MM-DD.md`
4. Meeting notes: `YYYY-MM-DD-topic.md`
5. Project specs: `project-name-spec.md`
6. Decision logs: `project-name-decision-001.md`

Avoid:
- `Final`, `v2`, `new`, `copy` suffixes
- mixed date formats (`2-4-26`, `Feb24`)
- duplicate names in different folders without qualifiers
