## Setup

```bash
# 1. Install dependencies
cd ~/clawd/skills/daily-devotional-auto
npm install

# 2. Copy environment template
cp .env.example .env

# 3. Edit .env with your keys:
# - NEWS_API_KEY (from newsapi.org)
# - ELEVENLABS_API_KEY (from elevenlabs.io)
# - YOUTUBE_CHANNEL_ID

# 4. Set up YouTube OAuth (one-time)
cd ~/clawd/skills/youtube-studio
node auth-full.js

# 5. Schedule with cron (runs daily at 9 AM)
0 9 * * * ~/clawd/skills/daily-devotional-auto/run-daily.sh
```

## User Suggestions

Viewers can suggest topics by commenting on videos with keywords like:
- "suggest", "topic", "pray for"
- "devotional about", "question about"
- "help with", "request"

The system will automatically detect and prioritize user suggestions for future videos.

## Video Format

Each video includes:
- Title with devotional theme
- "Generated by AI" subtitle
- User suggestion credit (if applicable)
- Date and scripture reference
- Blue waveform visualizer
- Your voice narration
