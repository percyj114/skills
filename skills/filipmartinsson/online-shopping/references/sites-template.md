# Store-Specific Notes

Add notes for stores as you use them. Example format:

```markdown
## example-store.com

- **Protection:** Cloudflare / none / other
- **Search URL:** `https://example-store.com/search?q=<query>`
- **Cart URL:** `https://example-store.com/cart`
- **Checkout URL:** `https://example-store.com/checkout`
- **Cookie banner:** Button text to dismiss
- **Checkout form field IDs:**
  - `#email` — email
  - `#phone` — phone
  - ...
- **Delivery options:** list radio button IDs
- **Payment options:** list radio button IDs
- **Notes:** language, quirks, restrictions
```
