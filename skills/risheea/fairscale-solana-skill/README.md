# FairScale Reputation Skill

Check Solana wallet reputation scores. 

- **Free tier:** 100 calls/day, no API key needed
- **Unlimited:** $0.01/call with prepaid credits

## Endpoint

```
https://x402.fairscale.xyz
```

## Quick Start

```bash
# Check a wallet score
curl "https://x402.fairscale.xyz/score?wallet=YOUR_WALLET"

# Check risk for a trade
curl "https://x402.fairscale.xyz/check?wallet=YOUR_WALLET&amount=500"
```

No API key required.
