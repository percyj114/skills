# Web应用安全审计报告模板

> 版本: 2.0.0  
> 支持两种模式：快速版（简洁）和完整版（详细）

---

## 使用说明

本模板提供两种报告模式：

- **快速版**：适合日常审计、CI/CD集成，包含核心漏洞信息和修复建议
- **完整版**：适合深度审计、渗透测试准备，包含详细的攻击链分析和测试验证

根据审计深度选择合适的章节填写。
