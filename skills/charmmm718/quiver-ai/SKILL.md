# QuiverAI Skill

QuiverAI is an AI-native vector graphics generation platform that creates high-quality SVG graphics from text prompts and reference images.

## Getting Started

### Web Interface
1. Visit https://app.quiver.ai/creations
2. Sign in or create an account
3. Enter a text prompt describing the desired graphic
4. Optionally upload reference images
5. Generate and download SVG

### API Usage
For programmatic access, see [API Reference](references/api.md).
