# AI店长 - 电商文案速查表

## 淘宝标题公式

```
[品牌] + [核心品类词] + [属性词1] + [属性词2] + [场景词] + [人群词] + [促销词]
```

示例：
- 北面 户外冲锋衣 男 防水透气 登山徒步 秋冬新款 加绒保暖
- 小熊 电热饭盒 上班族 便携 自热 保温 304不锈钢 插电加热

## 拼多多标题公式

```
[价格锚点词] + [核心词] + [长尾词1] + [长尾词2] + [属性堆叠]
```

示例：
- 特价清仓 露营折叠椅 户外便携式 超轻铝合金 靠背椅子 野餐钓鱼椅 沙滩椅 折叠凳
- 工厂直销 蓝牙耳机 无线 降噪 超长续航 运动跑步 入耳式 高音质 2026新款

## 小红书标题公式

```
[emoji] + [情绪钩子] + [产品关键词] + [效果/结果]
```

示例：
- 🏕️ 被这把露营椅种草了！轻到飞起颜值还高
- 😱 用了这个耳机再也回不去了...降噪绝了
- 💡 打工人必备！这个电热饭盒救了我的午餐

## 详情页五段式结构

```
1. 核心卖点（一句话，20字以内）
2. 痛点场景（没有这个产品时的困扰）
3. 解决方案（产品如何解决痛点）
4. 信任背书（材质/认证/销量/好评）
5. 行动号召（限时优惠/赠品/保障）
```

## 客服话术模板

### 欢迎语
- 亲，欢迎光临！有什么可以帮您的吗？😊
- Hi～感谢关注我们家！正在看哪款呢，我帮您介绍～
- 亲爱的，您好！今天有活动哦，有什么想了解的随时问我～

### 催付（温柔版）
- 亲，您选的这款库存不多了，怕您错过特意提醒一下～
- 亲，看到您还没付款，是遇到什么问题了吗？我帮您看看～
- 亲，今天下单还能赶上明天发货哦，需要我帮您备注什么吗？

### 差评挽回
- 亲，非常抱歉给您带来不好的体验！我们已经记录了您的问题，这边给您安排[补偿方案]，希望能弥补一下～
- 亲，看到您的评价我们很心痛。这个问题我们已经反馈给品控部门了。方便的话加一下我们的售后微信，给您寄一个新的可以吗？

### 砍价回复
- 亲，这个价格真的是最低了，我们利润很薄的😅 不过我可以帮您申请一个小赠品～
- 亲，价格确实没办法再低了，但是我可以帮您备注优先发货+送运费险，您看可以吗？

## 直播话术节奏

```
开场（30s）→ 展示（2min）→ 痛点（1min）→ 价格锚定（30s）→ 逼单（1min）→ 售后（15s）→ 感谢（15s）
```

### 价格锚定话术
- "这个在专柜要卖XXX，我们直播间今天只要XXX"
- "你们去搜一下同款，最便宜也要XXX，我们今天XXX拿走"
- "成本价XXX，今天不赚钱交个朋友，只要XXX"

### 逼单话术
- "最后30单了！抢完恢复原价！"
- "3、2、1，上链接！手慢无！"
- "已经有XXX人下单了，库存真的不多了"

## 违禁词速查

### 绝对不能用
最、第一、顶级、极致、绝对、万能、100%、永久、无敌

### 医疗相关（非药品不能用）
治疗、治愈、药效、疗效、消炎、杀菌（日用品）

### 虚假宣传
全网最低、史上最便宜、假一赔万、国家级、驰名商标

### 安全替代词
- 最好 → 优质/精选/热销
- 第一 → 领先/头部/热门
- 100%纯棉 → 高含棉量/精梳棉
- 永久 → 持久/长效
