# AI店长 (ai-dianzhang)

> 躺着开店，AI店长帮你搞定一切

你的 AI 电商助手。竞品监控、爆款文案、客服话术、数据日报——一个 Skill 搞定开店全流程。

## 能力一览

### 🔍 竞品监控
告诉店长你卖什么，它自动帮你：
- 搜淘宝、拼多多、1688、小红书的同类商品
- 分析价格区间、销量趋势、评价关键词
- 找出竞品的卖点和差评痛点
- 每天/每周自动推送竞品动态

```
店长，帮我看看淘宝上卖"露营折叠椅"的竞品情况
```

### ✍️ 爆款文案
丢一张图或简单描述，店长帮你写：
- 商品标题（适配淘宝/拼多多/小红书不同风格）
- 详情页文案（卖点提炼 + 场景化描述）
- 短视频脚本（带口播文案 + 分镜建议）
- 小红书种草笔记（标题党 + emoji + 标签）
- 直播话术（开场/讲品/逼单/感谢）

```
店长，帮我写一个露营折叠椅的淘宝标题，主打轻便和高颜值
```

### 💬 客服话术
根据你的商品自动生成：
- 售前FAQ（尺寸/材质/发货/优惠）
- 售后模板（退换货/物流/差评挽回）
- 催付话术（温柔提醒不惹人烦）
- 好评引导（不违规的巧妙话术）

```
店长，我卖的是蓝牙耳机，帮我生成一套客服话术
```

### 📊 数据日报
设定关注的品类或竞品，店长每天自动：
- 汇总行业热搜词变化
- 监控竞品价格调整
- 发现新上架的爆款
- 分析评价趋势（差评预警）

```
店长，每天早上9点帮我看看"无线充电器"这个品类的动态
```

### 🏷️ 选品建议
不知道卖什么？店长帮你分析：
- 当前热门品类和趋势
- 蓝海市场机会（竞争少、需求大）
- 季节性选品建议
- 1688 货源匹配

```
店长，最近有什么好卖的品类推荐？预算5万以内
```

### 📱 多平台适配
同一个商品，一键生成多平台内容：
- 淘宝/天猫：搜索优化标题 + 详情页
- 拼多多：低价引流标题 + 拼单文案
- 小红书：种草笔记 + 标签策略
- 抖音：短视频脚本 + 直播话术
- 1688：批发详情 + 工厂实力描述

```
店长，这个产品帮我出一套全平台文案
```

## 快速开始

安装后直接跟你的 OpenClaw 说：

```
店长，我想在淘宝开一家卖露营装备的店，帮我分析一下市场
```

店长会自动：
1. 搜索当前露营装备市场数据
2. 分析头部竞品的价格、销量、评价
3. 找出差异化机会点
4. 推荐具体选品方向
5. 给出定价建议

## 使用场景

| 你说 | 店长做 |
|------|--------|
| "帮我看看XX品类的竞品" | 搜索分析竞品数据，输出报告 |
| "帮我写个商品标题" | 生成多个标题方案，标注关键词 |
| "这个差评怎么回复" | 生成专业得体的回复话术 |
| "最近什么好卖" | 分析热门趋势，推荐蓝海品类 |
| "帮我出一套详情页文案" | 完整的卖点+场景+参数+FAQ |
| "每天帮我盯着竞品" | 设置定时监控，有变化就通知 |

## 工作原理

```
用户指令 → AI店长理解需求
                ↓
        搜索电商平台数据（web_search + web_fetch）
                ↓
        AI 分析（竞品/趋势/机会）
                ↓
        生成内容（文案/话术/报告）
                ↓
        定时任务（cron 监控 + 日报推送）
```

- 不需要你提供任何平台账号密码
- 不需要 API 授权
- 所有数据来自公开信息
- 用你配置的 LLM 生成内容（推荐配合 chinese-llm-router 使用国产模型，成本更低）
