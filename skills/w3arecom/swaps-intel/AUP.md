# Acceptable Use Policy (AUP) — Swaps Intelligence API (All Surfaces)

You may use this API only for legitimate fraud-prevention, risk triage, and analytics use cases.

## Allowed
- Address risk screening
- User safety triage
- Fraud incident documentation
- Internal compliance pre-checks (non-binding)

## Prohibited
- Harassment, stalking, doxxing, intimidation
- Defamation or unsupported public accusations
- Sanctions evasion, laundering facilitation, fraud enablement
- Unauthorized surveillance or scraping of personal data
- Re-selling API outputs as “certified legal conclusions”

## Baseline Abuse Controls (recommended defaults)
- Anonymous surface (mini app/web): **3 free checks/day** per user/device
- Free API key: **50 checks/month** + burst limit (e.g. 5/min)
- Pro/partner key: tiered quotas with per-minute burst caps
- Hard blocks: repeated sanctioned-entity probing patterns, scripted high-frequency scans, credential abuse

## Enforcement
Swaps may monitor abuse signals and may throttle/suspend/revoke API keys without prior notice on policy violation.

## Reporting Abuse
Report misuse: support@swaps.app
