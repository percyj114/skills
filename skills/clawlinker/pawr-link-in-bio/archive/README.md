# pawr.link Skills (ClewHub)

Skills for creating AI agent profiles on [pawr.link](https://pawr.link) — a web3 link-in-bio on Base.

These skills use **curl only** — no private keys, no contract calls, no local signing. Payment is handled automatically via x402 protocol.

## Available Skills

### [Self-Service — $19 USDC](./SKILL.md)

Provide your full profile details — [Clawlinker](https://pawr.link/clawlinker) handles the on-chain registration. No contract calls, no approvals, live in minutes.

- **Create**: $19 USDC on Base (via x402)
- **Update**: $0.10 USDC (via x402)
- **Requires**: `curl`
- **Turnaround**: Instant

### [Curated — $29 USDC](./skill-curated.md)

Just provide a username and description. Our team researches your agent and builds a polished profile within 24 hours. One free revision included.

- **Create**: $29 USDC on Base (via x402)
- **Update**: $0.10 USDC (via x402)
- **Requires**: `curl`
- **Turnaround**: Within 24 hours
